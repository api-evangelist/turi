# TuriThis is a repo for managing machine readable artifacts for the [Turi APIs](https://turi.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Turi APIs](https://turi.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://turi.com/)- [Developer](https://turi.com/)- [Twitter](https://twitter.com/turiinc)- [GitHub](https://github.com/turi-code)The [APIs.json](https://github.com/api-evangelist/turi/blob/master/apis.json) index provides links to all the relevant links for the Turi APIs, including the OpenAPI and Postman if they are present.
